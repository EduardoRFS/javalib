Documentation

If you need documentation, you can:

  • read the on-line tutorial
  • read the on-line API documentation
  • make the documentation (see INSTALL file) and read starting from javalib/doc/api/index.html
  • browse the .mli files in the source
  • read the Java Virtual Machine Specifications for more information about the class file format
  • send an email to Laurent Hubert <first.last@irisa.fr>

Javalib is distributed with an implementation of the Patricia trees. The documentation is in the interface files in javalib/ptrees/.
