Installation

You need ExtLib (>=1.5.1) and CamlZip (>=1.03).

ExtLib

To build ExtLib (if the version you download is 1.5.1):

  • tar zxf extlib-1.5.1.tar.gz
  • cd extlib-1.5.1
  • make all opt

If you have ocamlfind, you can install ExtLib with:

  • sudo make install

Otherwise, you can install ExtLib with:

  • sudo mkdir `ocamlc -where`/extlib
  • sudo cp *.{cmi,cma,a,mli,cmxa} `ocamlc -where`/extlib

If you install ExtLib like this, you will have to edit the Makefile.config file in javalib's directory and replace +site-lib/extlib with +extlib.

CamlZip

To build and install CamlZip (if the version you download is 1.03):

  • tar zxf camlzip-1.03.tar.gz
  • cd camlzip
  • make all allopt
  • sudo make install installopt

Javalib

To build and install Javalib (if version is 2.0.1):

  • tar jxf javalib-2.0.1.tar.bz2
  • cd javalib-2.0.1
  • make all
  • sudo make install

Note: If something went wrong at some point, you might have to edit Makefile.config just before compiling Javalib.

If you want the documentation:

  • make doc
