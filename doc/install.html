Installation

You need Findlib (>=1.2.3), ExtLib (>=1.5.1) and CamlZip (>=1.03).

ExtLib

To build and install ExtLib (if the version you have downloaded is 1.5.1):

  1. tar zxf extlib-1.5.1.tar.gz
  2. cd extlib-1.5.1
  3. sed -i 's/"1.5"/"1.5.1"/' META.txt
  4. make all opt
  5. sudo make install

Note: Step 3 fixes a bug in the version number stored in the META file of ExtLib.

CamlZip

To build and install CamlZip (if the version you have downloaded is 1.04):

  1. tar zxf camlzip-1.04.tar.gz
  2. cd camlzip
  3. make all allopt
  4. sudo make install installopt

Javalib

To build and install Javalib (if the version you have downdloaded is 2.0.1) (you may skip Step 3 if you installed CamlZip using Findlib - either manually or with a package manager):

  1. tar jxf javalib-2.0.1.tar.bz2
  2. cd javalib-2.0.1
  3. sudo make -C camlzip
  4. make ptrees
  5. sudo make installptrees
  6. make all
  7. sudo make install

Note: If something went wrong at some point, you might have to edit Makefile.config.

If you want the documentation:

  • make doc
