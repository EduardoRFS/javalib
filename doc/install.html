Installation

You need Findlib (>=1.2.3), ExtLib (>=1.5.1) and CamlZip (>=1.03) (provided in the archive).

ExtLib

To build and install ExtLib (if the version you have downloaded is 1.5.1):

  1. tar zxf extlib-1.5.1.tar.gz
  2. cd extlib-1.5.1
  3. sed -i 's/"1.5"/"1.5.1"/' META.txt
  4. make all opt
  5. sudo make install

Note: Step 3 fixes a bug in the version number stored in the META file of ExtLib.

Javalib

To build and install Javalib (if the version you have downdloaded is 2.0.1) (you may skip Step 3 and 4 if you already have installed CamlZip using Findlib - either manually or with a package manager):

  1. tar jxf javalib-2.0.1.tar.bz2
  2. cd javalib-2.0.1
  3. make -C camlzip
  4. sudo make -C camlzip install
  5. make -C ptrees
  6. sudo make -C ptrees install
  7. make all
  8. sudo make install

Note: If something went wrong at some point, you might have to edit Makefile.config.

If you want the documentation:

  • make doc
